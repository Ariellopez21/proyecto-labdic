\section{Objetivos}

\subsection{Objetivo general}
Describir la implementación completa del módulo de usuarios, asegurando su correcta integración tanto con el backend como con el frontend, destacando que al final del hito se debe contar con un módulo robusto que sirva como referencia para implementar los módulos restantes (productos y préstamos).

\subsection{Objetivos específicos}
\begin{itemize}
	\item Definir la arquitectura del módulo de usuarios de manera que sea modular, escalable y mantenible.
	
	\item Implementar el backend del módulo de usuarios usando Litestar para exponer API RESTful, SQLAlchemy y Alembic para el acceso a datos, y librerías modernas para autenticación como \texttt{pwdlib}.
	
	\item Diseñar e implementar el frontend del módulo de usuarios usando Vue 3 y su ecosistema (Composition API, Pinia, Vue Router, PrimeVue, Vite y TailwindCSS). Adicionalmente, se debe explicar cómo cada herramienta contribuye a la modularidad, al mantenimiento y a la experiencia de usuario.
	
	\item Establecer la comunicación entre frontend y backend mediante interfaces tipadas (TypeScript) y buenas prácticas de consumo de API.
	
	\item Dejar una base preparada para la implementación de los módulos de productos y préstamos, estableciendo criterios de escalabilidad que se profundizarán en el Hito 4.
\end{itemize}
